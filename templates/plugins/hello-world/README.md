# Hello World Plugin (Template)

This is a starter plugin for OpenWeb.

## Features

- Adds API route: `GET /api/plugins/hello-world/ping`
- Creates plugin table: `plugin_hello_world_plugin_visits`
- Registers cron job every 5 minutes
- Loads a simple client script in the public site

## Install

1. Zip this folder.
2. In OpenWeb Admin: `System -> Plugins`.
3. Upload the ZIP.
4. Enable plugin.
5. Reload API from the plugin row.
